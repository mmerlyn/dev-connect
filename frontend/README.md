# DevConnect Frontend

React frontend for DevConnect. See the [main README](../README.md) for full documentation.

## Quick Start

```bash
npm install
npm run dev
```

Runs on http://localhost:5173

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
| `npm run lint` | Run ESLint |
